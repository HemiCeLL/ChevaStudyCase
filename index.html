About RecipeHub

RecipeHub adalah platform untuk menemukan resep makanan dari seluruh dunia. Kami menggunakan TheMealDB API untuk memberikan Anda akses ke ribuan resep yang lezat dan mudah diikuti.

Setiap resep dilengkapi dengan bahan-bahan, petunjuk memasak, dan video tutorial untuk memudahkan Anda menciptakan hidangan yang sempurna.

Clean Code

Peserta diharapkan menulis kode dengan struktur yang rapi dan terorganisir.

  • Penamaan yang Konsisten

    Class, function, dan variable harus mewakili apa yang dibuat (misal: var age = 10)

  • Struktur Folder Terstruktur

    Letak foldering jelas dan tidak redundant, sesuai kebutuhan project

  • Semantic HTML

    Menggunakan tag HTML yang bermakna dan sesuai konteks

  • Hindari Duplikasi

    Menghindari duplikasi atau hardcode yang tidak diperlukan
