   - Petunjuk memasak
   - Link tutorial video YouTube
   - Link sumber resep

- Atau klik area gelap di luar modal
- Atau tekan tombol Escape

## ğŸ”§ Troubleshooting

### "No recipes found"
**Solusi:**
- Cek ejaan pencarian
- Coba dengan kata kunci lain
- Cek koneksi internet

### "Loading spinner tidak berhenti"
**Solusi:**
- Cek koneksi internet
- Refresh halaman (F5)
- Coba browser lain

### "Video tidak bisa dimainkan"
**Solusi:**
- Klik link "Watch Tutorial"
- Akan membuka YouTube di tab baru
- Pastikan YouTube tidak diblokir

### "Modal tidak terbuka"
**Solusi:**
- Refresh halaman
- Coba klik resep lain
- Coba browser lain
